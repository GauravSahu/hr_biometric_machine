# Some Usefull Function of ZKLIB

1. To Connect Machine 
<pre>
zk = zklib.ZKLib(machine_ip, int(port))
res = zk.connect()
</pre>
2. To Disconnect Machine
<pre>
zk.disconnect()
</pre>
Example:
<pre>
zk = zklib.ZKLib(machine_ip, int(port))
res = zk.connect()
if res == True
	zk.disconnect()
</pre>
3. To Enable  Device
<pre>
zk.enableDevice()
</pre>
Example:
<pre>
zk = zklib.ZKLib(machine_ip, int(port))
res = zk.connect()
if res == True
	zk.enableDevice()
</pre>
4. To Disable  Device
<pre>
zk.disableDevice()
</pre>
Example:
<pre>
zk = zklib.ZKLib(machine_ip, int(port))
res = zk.connect()
if res == True
	zk.enableDevice()
	zk.disableDevice()
</pre>

5. To Get Device Version
<pre>
zk.version()
</pre>
Example:
<pre>
zk = zklib.ZKLib(machine_ip, int(port))
res = zk.connect()
if res == True
	print zk.enableDevice()
	print zk.disableDevice()
	print zk.version()
</pre>

6. To Get Device OS Version
<pre>

Example:
<pre>
zk = zklib.ZKLib(machine_ip, int(port))
res = zk.connect()

8. To Download Attendance
<pre>

</pre>
Example:
<pre>
